# Example1d - MOOSE/SAM Plugin

## Purpose

This example provides a demonstration on how to use WATTS to perform multiple runs and save selected outputs from all runs to a single '.csv' file for ease of comparison, post-processing, and visualization.

## Code(s)
 
- SAM

## Keywords
 
- Parametric study
- Results extraction
- Post-processing

## File descriptions

- example1d.py: WATTS workflow for this example. This is the file to execute to run the problem described above.
- sam_template: SAM templated input file.
- results.csv: An example of CSV file that will be generated through running this example.
